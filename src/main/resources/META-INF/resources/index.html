Filters

Example Expression Description
src/main/java/** Matches all files and subdirectories under src/main/java.
**/*.java,**/*.kt Matches every .java or .kt file.
some/*/*/path/** Matches files in paths with exactly 2 directory levels between some and path.
**/target/** Matches any target folder and its contents (may be useful for exclusion).
regex:.*Test\.java Regluar expression that matches all Java test files (ending in Test.java) anywhere.
